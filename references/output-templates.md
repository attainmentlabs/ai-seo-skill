## 4. Citation Hook Examples

When writing citation-worthy paragraphs, use these formats:

### Statistic Hook
```
[Specific number] [specific outcome] according to [source] ([year]). This represents [context or comparison]. For [audience], this means [implication].
```
**Example:** "87% of ChatGPT citations match pages ranking in Bing's top 10 according to SE Ranking (2025). This confirms that traditional SEO remains the foundation for AI visibility. For businesses investing in GEO, this means SEO is prerequisite infrastructure, not a competing priority."

### Definition Hook
```
[Term] is [clear definition in one sentence]. Unlike [related concept], [term] specifically [distinguishing characteristic]. The term was [origin/attribution].
```
**Example:** "AIVO (AI Visibility Optimization) is the systematic practice of monitoring and correcting how AI systems represent a brand. Unlike GEO which focuses on citation acquisition, AIVO assumes the brand is already being mentioned and ensures accuracy. The framework was formalized in an August 2025 VML paper."

### Framework Hook
```
The [framework name] consists of [number] components: [list]. Each component addresses [specific problem]. When applied together, [outcome].
```

### Comparison Hook
```
[Option A] [key metric]: [value]. [Option B] [same metric]: [value]. The [X%] difference is attributed to [factor]. For [use case], [recommendation].
```

### Authority Hook
```
According to [credentialed source], "[direct quote]." This [validates/challenges] [common assumption]. [Source credentials: title, organization, relevant experience].
```

## 5. Schema Code Templates

### FAQ Schema
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question text]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer text, 40-60 words ideal]"
      }
    }
  ]
}
```

### Organization Schema
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "[Business Name]",
  "url": "[Website URL]",
  "logo": "[Logo URL]",
  "sameAs": [
    "[LinkedIn URL]",
    "[Twitter URL]"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "[Phone]",
    "contactType": "customer service"
  }
}
```

### Person/Author Schema
```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "[Full Name]",
  "jobTitle": "[Title]",
  "worksFor": {
    "@type": "Organization",
    "name": "[Company]"
  },
  "sameAs": [
    "[LinkedIn URL]"
  ],
  "knowsAbout": ["[Expertise 1]", "[Expertise 2]"]
}
```

### LocalBusiness Schema
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "[Business Name]",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "[Street]",
    "addressLocality": "[City]",
    "addressRegion": "[State]",
    "postalCode": "[ZIP]"
  },
  "telephone": "[Phone]",
  "openingHoursSpecification": [
    {
      "@type": "OpeningHoursSpecification",
      "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"],
      "opens": "09:00",
      "closes": "17:00"
    }
  ]
}
```
