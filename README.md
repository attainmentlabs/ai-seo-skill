# AI SEO Skill for Claude Code

Traditional search engine optimization across 13 disciplines. Get ranked on Google, Bing, YouTube, App Stores, marketplaces, and every major search platform.

## What's Included

- **13 Search Disciplines:** SEO, AEO, Entity SEO, Semantic SEO, Voice Search, Video SEO, Visual Search, Social Search, AI Local SEO, AI Overviews, ASO, Podcast/Audio, Marketplace Search
- **13 Reference Files:** Deep-dive guides for each discipline
- **Business Type Prioritization:** Which disciplines matter most for SaaS, E-commerce, Local Services, Healthcare, Professional Services, Media, Hospitality, Education
- **Quick-Start Playbook:** 4-week implementation plan
- **Output Templates:** Audit scorecards, 90-day plans, schema code

## Installation

1. Download `SKILL.md` and the `references` folder
2. Drag and drop into Claude (Desktop app, VS Code, Cursor, or Terminal)
3. Tell Claude: "Install this skill"
4. Invoke with `/ai-seo-skill`

## Usage

```
/ai-seo-skill
```

Then ask Claude anything:
- "Audit my website's SEO across all channels"
- "What SEO disciplines should a local dental practice prioritize?"
- "Optimize this page for featured snippets"
- "Generate schema markup for my business"
- "How do I rank in YouTube search?"
- "Optimize my app store listing"

## The 13 Disciplines

### Tier 1: Core (Every Business)
1. **SEO** - Google/Bing organic rankings
2. **AEO** - Featured snippets, People Also Ask, position zero
3. **Entity SEO** - Knowledge Graph, Wikidata recognition
4. **Semantic SEO** - Topic clusters, topical authority

### Tier 2: High-Value Channels
5. **Voice Search** - Siri, Alexa, Google Assistant
6. **Social Search** - TikTok, Instagram, Pinterest, LinkedIn
7. **Video SEO** - YouTube, Google Video results
8. **AI Local SEO** - Google Maps, Apple Maps, ChatGPT local
9. **Visual Search** - Google Lens, Pinterest Lens

### Tier 3: Specialized
10. **AI Overview Optimization** - Google AI summaries
11. **ASO** - App Store search rankings
12. **Podcast/Audio Search** - Apple Podcasts, Spotify
13. **Marketplace Search** - Amazon, Walmart AI
